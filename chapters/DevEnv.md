# Development Environment

## Hardware requirements

| Requirement         | Minimum | Recommended |
|---------------------|---------|-------------|
| CPU Cores           | 4       | 8           |
| Memory(GB)          | 8       | 16          |
| Free disk space(GB) | 44.0    | 70.0        |

## Software requirements

* Operating system
    * macOS 14+
    * Windows(?)
* Dev Tools
    * Git 2.27+
    * JDK 17+
    * Flutter SDK
* Android
    * Android Studio 2023.1 (Hedgehog)+
    * Flutter plugin for IntelliJ
* iOS
    * Xcode 15+
    * CocoaPods 1.13+
